# Use Cases — AI-Powered Banking & Financial Assistant

---

## 1. Overview

This document defines key real-world use cases for the AI-powered financial assistant. These use cases demonstrate how users interact with the system to solve financial problems using natural language.

---

## 2. Use Case 1: Personal Spending Analysis

### Actor:

Retail Banking Customer

### Scenario:

User wants to understand where their money is being spent.

### Input:

"Where am I spending most of my money?"

### System Behavior:

* Analyze user-provided expense data (or simulated dataset)
* Categorize expenses (food, rent, travel, etc.)
* Identify highest spending category

### Output:

* Summary of spending
* Top expense categories
* Suggestions to reduce unnecessary spending

### Business Value:

* Improves financial awareness
* Encourages better budgeting habits

## 3. Use Case 2: Savings Recommendation

### Actor:

Retail User

### Scenario:

User wants to save more money but doesn’t know how.

### Input:

"How can I save more money with my current salary?"

### System Behavior:

* Analyze income vs expenses
* Suggest savings strategies
* Recommend percentage-based savings plan

### Output:

* Monthly savings target
* Expense reduction tips
* Budget allocation plan

### Business Value:

* Increases customer trust
* Encourages financial discipline

## 4. Use Case 3: Loan & EMI Guidance

### Actor:

Loan Applicant

### Scenario:

User wants to understand loan options and EMI breakdown.

### Input:

"I want a home loan of 20 lakh. What will be my EMI?"

### System Behavior:

* Calculate EMI (based on assumed interest rate)
* Explain breakdown (principal + interest)
* Suggest optimal tenure

### Output:

* EMI amount
* Total interest payable
* Recommended plan

### Business Value:

* Reduces dependency on loan officers
* Improves loan conversion rate

## 5. Use Case 4: Investment Suggestions (Basic)

### Actor:

Beginner Investor

### Scenario:

User wants simple investment advice.

### Input:

"Where should I invest 50,000 rupees?"

### System Behavior:

* Understand risk profile (basic assumption)
* Provide general investment options

  * Fixed deposits
  * Mutual funds (basic suggestion)
  * Savings instruments

### Output:

* Investment options with pros/cons
* Risk explanation

### Business Value:

* Improves engagement
* Positions platform as financial advisor
